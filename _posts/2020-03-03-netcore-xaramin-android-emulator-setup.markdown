Enable below 3 windows feature
```
   1. Hyper-V
   2. Windows Hypervisor Platform
   3. Windows Sandbox
```

Fix any Android SDK Tools from Visual 2019
```
Tools -> Android -> Android SDK Manager
```

Platform
```
AMD Ryzen 2700X, Windows 10
Visual Studio Community 2019
```
