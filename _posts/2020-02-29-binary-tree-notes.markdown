# Depth-First Search
---
- PreOrder Traversal

  Root -> Left-Leaf -> Right-Leaf

- InOrder Traversal

  Left-Leaf -> Root -> Right-Leaf

- PostOrder Traversal

  Left-Leaf -> Right-Leaf -> Root
